# 📱 ScholarOS Mobile - تطبيق الهاتف المحمول المصاحب لمنصة الطالب الأكاديمية

تطبيق الهاتف المحمول المصاحب لمنصة **ScholarOS**، تم بناؤه وتطويره بالكامل باستخدام **Expo (React Native)** مع لغة **TypeScript** ولغة واجهة مستخدم ناطقة بالكامل باللغة العربية تدعم اتجاه الكتابة من اليمين إلى اليسار (RTL).

يعمل التطبيق بنظام **محلي أولاً (Local-First)**، حيث يحتفظ بقاعدة بيانات مستقلة على جهاز الطالب وهاتفه المحمول، ويقوم بمزامنتها بشكل فائق الأمان عبر تشفير طرفي مع تطبيق سطح المكتب من خلال سحابة **Supabase**.

---

## 🛠️ البنية التقنية وهيكل التطبيق (Tech Stack)

* **Expo (React Native v56)**: الإطار الأساسي لبناء تطبيق هاتف هجين سريع وخفيف يدعم التصدير الفوري لنظامي Android و iOS.
* **Expo SQLite**: قاعدة بيانات خفيفة وسريعة علائقية تعمل محلياً بالكامل على قرص الهاتف لتخزين المواد، جداول الحصص، المهام، والدرجات.
* **Crypto-JS**: محرك تشفير وفك تشفير متكامل يطبق خوارزمية **AES-256-CBC** وتوقيع رقمي **SHA-256** بطريقة متطابقة تماماً مع كود التشفير لنسخة سطح المكتب، لضمان تبادل السجلات المشفّرة بنجاح وفك تشفيرها فقط بمفتاح الطالب الخاص.
* **AsyncStorage**: لتخزين تفضيلات الطالب وإعدادات الاتصال السحابي (Supabase URL, Anon Key).
* **Lucide React Native**: لتوفير حزمة أيقونات أكاديمية وتفاعلية متناسقة بصرية تماثل جمالية سطح المكتب.

---

## 🔒 بنية المزامنة والتشفير الآمن (SyncGuard Mobile Engine)

يعتمد التطبيق في تبادل البيانات على بروتوكول **SyncGuard** المطور:

1. **طابور التعديلات (sync_queue)**: تُسجل كافة التعديلات المحلية (إضافة، تعديل، حذف منطقي) كعمليات معلقة محلياً.
2. **التشفير الطرفي**: تُشفّر حقول البيانات كاملة باستخدام رمز العبور الخاص بالطالب بصيغة AES-256-CBC، مما ينتج رمز تشفير معزولاً ورمز تحقق SHA-256.
3. **الاتصال السحابي (Push & Pull)**: يقوم التطبيق بشكل دوري (أو يدوي بضغطة زر) برفع التعديلات الجديدة لسحابة Supabase وسحب التعديلات التي أجراها الطالب على حاسوبه الشخصي.
4. **حل التعارضات الأكاديمية (Rules 1-5)**: يقوم التطبيق بتصفية البيانات تلقائياً وتطبيق خوارزميات دمج الجداول، الحذف المنطقي، تكرار التوقيت، وحفظ درجات المقررات للمراجعة لضمان سلامة بياناتك.

## 📋 شاشات التطبيق الجوهرية:

* 📊 **الرئيسية (Dashboard)**: ترحيب أكاديمي دافئ يعرض محاضرات اليوم، عدد المهام المتبقية، والمعدل التراكمي الفصلي.
* 📅 **الجدول الدراسي (Timetable)**: مستعرض مهام جدول الحصص مقسم أيام الأسبوع (من السبت للخميس) يظهر أوقات المحاضرات والقاعات والمحاضرين، مع إمكانية التعديل والإضافة المباشرة.
* ✏️ **المهام والأجندة (Tasks Checklist)**: مستعرض تفاعلي لتسجيل متطلبات ومشاريع المواد الدراسية مقسم لمهام معلقة ومهام منجزة تدعم الشطب الفوري.
* 📈 **حاسبة المعدل والدرجات (GPA Tracker)**: حاسبة ذكية ترصد درجاتك محلياً وتقوم باحتساب المعدل التراكمي الفصلي والعام مباشرة بناءً على أوزان وساعات المقررات.
* 🔄 **المزامنة والإعدادات (Settings Panel)**: شاشة تحرير إعدادات السحابة، محاكي وضع المطور (أوفلاين)، واستعراض تقارير المزامنة الأخيرة.

## 🚀 دليل البدء المحلي والتشغيل (Getting Started)

### 📋 المتطلبات الأساسية:
* تثبيت [Node.js](https://nodejs.org/) (الإصدار 18 أو أحدث).
* هاتف محمول مثبت عليه تطبيق **Expo Go** (متوفر مجاناً على متجر Google Play ومتجر Apple App Store).

### 🛠️ خطوات التشغيل الفورية:

1. **الانتقال لمجلد تطبيق الهاتف**:
   ```bash
   cd scholar-os-mobile
   ```

2. **تثبيت مكتبات وأدوات التشغيل**:
   ```bash
   npm install
   ```

3. **بدء تشغيل خادم التطوير (Metro Bundler)**:
   ```bash
   npm start
   ```

4. **المعاينة على هاتفك الشخصي**:
   * قم بفتح تطبيق **Expo Go** على هاتفك.
   * امسح رمز الاستجابة السريعة (QR Code) الظاهر أمامك في سطر الأوامر لمعاينة وتجربة التطبيق مباشرة على شاشة هاتفك مع خاصية التحديث الفوري للكود (Hot Reload)!

## 📦 تجميع وتصدير ملف التثبيت المستقل (.apk للـ Android):

لتصدير تطبيق الهاتف بالكامل في ملف تثبيت نهائي للـ Android لتثبيته بشكل دائم على هاتفك دون الحاجة للـ Expo Go، يمكنك تشغيل أوامر **EAS Build** التالية:

```bash
# 1. تثبيت واجهة الأوامر السحابية لـ Expo
npm install -g eas-cli

# 2. تسجيل الدخول لحساب Expo الخاص بك
eas login

# 3. تجميع ملف الـ APK سحابياً (سيعطيك رابطاً مباشراً لتحميل الملف فور انتهائه)
eas build --platform android --profile preview
```
